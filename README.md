# 🚀 Modern SaaS Landing Page

最新のITツールやアプリの紹介ページに最適な、Framer風の美しいSaaSランディングページ・テンプレートです。

**👉 [ライブプレビューを見る (Live Demo)](https://kotao-boop.github.io/framer-template-saas/)**

## ✨ 特徴
- 画面中央の目を引くグラデーションテキストとオーロラ背景。
- 洗練されたナビゲーションと、滑らかなアニメーションを備えた「Glow Button（光るボタン）」。
- 複雑なフレームワーク（ReactやNext.js）不要で、たった1つのHTMLファイルで完全に動作します。

## 🚀 使い方
1. `index.html` 内のテキストやボタンのリンク先を自由に変更してください。
2. カラーパレットは `<style>` タグ内の `:root` 変数を変更するだけで一括でテーマカラーを変えられます。
